# FunCaption - AI Instagram Caption Generator

AI-powered Instagram caption generator with regional vibes and belonging psychology.

## Features

- 30 Moods for every emotion
- 18 Regional Vibes (Gujarati, Punjabi, Tamil, etc.)
- Multiple pricing plans (Free to Business)
- Razorpay payment integration
- Google Gemini AI powered
- Conversion-optimized UI with psychology tactics

## Tech Stack

- *Frontend:* Next.js 13, React, Tailwind CSS
- *Backend:* Next.js API Routes
- *Database:* Supabase (PostgreSQL)
- *AI:* Google Gemini API
- *Payments:* Razorpay
- *Hosting:* Vercel
